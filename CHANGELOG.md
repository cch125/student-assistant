# 更新日志

本项目采用语义化版本号。每次大更新都会保留 Git 提交历史、创建版本标签并推送到 GitHub，因此可以随时查看或恢复之前的版本。

## [0.4.0] - 2026-07-19

### 新增

- 服务卡片生成阶段自动提取暨南大学官方页面中的 PDF、Word、Excel 和压缩包附件直链。
- 助手回答优先提供“直接下载”按钮，同时保留“查看官方说明”入口。
- 通知、办事指南等没有附件的内容只显示官方说明，不会误标为可下载文档。

### 修正

- 提高标题和材料关键词精确命中的优先级，避免下载问题返回语义相近但错误的文档。
- 对需要统一身份认证、在线生成或已失效的附件明确提示，不提供不可用下载按钮。

## [0.3.0] - 2026-07-19

### 新增

- 建立正式版本管理规则，加入 `VERSION`、更新日志和发布检查脚本。
- 数据流程看板显示 RAGFlow 各知识库的文档、分块及解析成功、处理中、失败数量。
- 看板明确区分“知识库文档解析状态”和“RAGFlow 数据管道日志”，避免把空日志误认为空知识库。

### 数据状态

- 核心服务卡片知识库：34 个文档，37 个分块。
- 第一阶段综合知识库：已补充第二阶段学生事务内容，覆盖本科休学、复学、退学、研究生服务、就业手续和公费医疗等事项。

## [0.2.0] - 2026-07-19
